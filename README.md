**HR Analysis Project**

📝 Overview
This project focuses on analyzing employee turnover, specifically addressing the "Brain Drain" phenomenon (the loss of high-performing talent). The dashboard identifies key departments and age groups at risk, providing data-driven insights to help HR teams improve retention strategies.

📊 Dataset
Source: Employee HR Record dataset.

Key Features: Employee ID, Department, Age, Performance Rating, Satisfaction Score, Hire/Exit Dates, and Business Impact (Sales).

🛠 Tools & Technologies
Python (Pandas): Data cleaning and Exploratory Data Analysis (EDA).

SQL (PostgreSQL/SQL Server): Querying employee data and calculating turnover metrics.

Power BI: Data modeling, DAX (Calculated Columns/Measures), and Interactive Visualization.

🚀 Steps Performed
Data Cleaning: Handled missing values (Exit Dates) and standardized department names using Python.

SQL Analysis: Wrote queries to calculate Turnover Rate and Average Tenure.

Data Modeling (DAX):

Created Status column (Current vs. Leaver).

Developed Age_Group with custom sorting logic.

Calculated Avg_Performance_Rating and Turnover Rate %.

Dashboard Design: Built a 2-page interactive report focusing on performance-to-satisfaction correlation.

📈 Key Results (Insights)
Brain Drain Confirmed: High-performers (Rating > 3.5) in "Store Operations" and "Customer Service" have a higher turnover rate than average employees.

Gen Z & Millennial Risk: Employees aged 26-35 show the highest resignation rate (20.62%).

Flight Risk Identification: The "Flight Risk" scatter plot pinpointed top-tier employees with low satisfaction scores, allowing for proactive retention.
